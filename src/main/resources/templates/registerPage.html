Rejestracja

Imię musi spełniać następujące warunki:
  • zaczyna się wielką literą
  • zawiera tylko litery
Nazwisko musi spełniać następujące warunki:
  • maksymalnie 2 człony
  • każdy człon zaczyna się wielką literą
Ulica musi spełniać następujące warunki:
  • zaczyna się wielką literą
  • zawiera tylko litery
  • maksymalnie 2 człony
Numer domu musi spełniać następujące warunki:
  • na końcu może być litera
  • zapisany może być jako ułamek
Numer lokalu musi spełniać następujące warunki:
  • większy od 0
  • jest liczbą
Kod pocztowy musi spełniać następujące warunki:
  • zawiera tylko cyfry
Miejscowość musi spełniać następujące warunki:
  • zaczyna się wielką literą
  • zawiera tylko litery
  • maksymalnie 2 człony
Telefon musi spełniać następujące warunki:
  • zawiera poprzedający numer kierunkowy np. +48
  • zawiera 9 cyfr numeru telefonu
Hasło musi spełniać następujące warunki:
  • długość od 6 do 25
  • zawiera jedną mała literę
  • zawiera jedną wielką literę
  • zawiera jedną cyfrę
  • zawiera jeden znak specjalny
Powtórzone hasło musi spełniać następujące warunki:
  • takie same jak pole hasło
